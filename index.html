NSMoment

Parse, validate, manipulate, and formatte dates easily in Objective-C

  • Read the Doc
  • Version 0.1.0

NSMoment is a library which provides an high abstraction level for parsing, validating, manipulating, and formatting the dates in Objective-C.

Its API is inspired by the well known moment.js library, however unlike its counterpart, its core is built upon the Foundation Framework components (NSDate, NSCalendar, etc.) to enable the interoperability with them.

This library is designed to facilitate the manipulation of times, dates, calendars, and durations in Objective-C by providing a single, easy, and unified approach to dealing with them.

Formatting Dates

          
// Display: October 27 2013, 10:49:48 AM
NSLog(@"%@", [[NSMoment now] format:@"MMMM dd yyyy, h:mm:ss a"]);

// Display: Oct 27 13
NSLog(@"%@", [[NSMoment now] format:@"MMM dd yy"]); 

// Display: 2013 escaped 2013
NSLog(@"%@", [[NSMoment now] format:@"yyyy 'escaped' yyyy"]);

// Display: 2013-10-27T10:49:48+0100
NSLog(@"%@", [[NSMoment now] format]);

Relative Times

          
// Display: 2 years ago
NSLog(@"%@", [[NSMoment momentWithDateAsString:@"20111031"] fromNow]); 

// Display: a year ago
NSMoment *m = 
[NSMoment momentWithDateAsString:@"2012/06/20" format:@"yyyy/MM/dd"];
NSLog(@"%@", [m fromNow]); 

// Display: 19 hours ago
NSLog(@"%@", [[[NSMoment now] startOf:@"day"] fromNow]); 

// Display: in 5 hours
NSLog(@"%@", [[[NSMoment now] endOf:@"day"] fromNow]); 

// Display: 11 minutes ago
NSLog(@"%@", [[[NSMoment now] startOf:@"hour"] fromNow]);

Languages

NSMoment currently supports Albanian, English, French and Spanish for relative times and all the native languages for the formatters.

          
// Display: il y a quelques secondes
NSMoment *french = [[NSMoment now] addAmountOfTime:-3 forUnitKey:@"s"];
[french setLocale:[NSLocale localeWithLocaleIdentifier:@"fr_FR"]];
NSLog(@"%@", [french fromNow]);

// Display: disa sekonda me parÃ«
NSMoment *albanian = [[NSMoment now] addAmountOfTime:-3 forUnitKey:@"s"];
[albanian setLocale:[NSLocale localeWithLocaleIdentifier:@"sq_AL"]];
NSLog(@"%@", [albanian fromNow]); 

// Display: hace unos segundos
NSMoment *spanish = [[NSMoment now] addAmountOfTime:-3 forUnitKey:@"s"];
[spanish setLocale:[NSLocale localeWithLocaleIdentifier:@"es_ES"]];
NSLog(@"%@", [spanish fromNow]);

// Display: in 11 months
NSMoment *reference = [NSMoment momentWithArray:@[@2013]];
NSMoment *english   = [[NSMoment now] addAmountOfTime:-3 forUnitKey:@"s"];
[english setLocale:[NSLocale localeWithLocaleIdentifier:@"en_US"]];
NSLog(@"%@", [english fromMoment:reference]);
